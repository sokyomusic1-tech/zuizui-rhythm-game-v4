# Zuizui Rhythm - リズムゲームアプリ

オリジナル曲「ズイズイソング和風ロック」を使用したスマートフォン向けリズムゲームアプリです。

## 特徴

- **3段階の難易度**: Easy、Normal、Hardの3つの難易度レベル
- **和風テーマ**: 朱色をアクセントカラーとした和風デザイン
- **ハイスコア記録**: 各難易度のハイスコアをローカルに保存
- **タップ判定システム**: Perfect、Good、Missの3段階判定
- **コンボシステム**: コンボ数に応じてスコア倍率が上昇
- **ハプティクスフィードバック**: タップ時の触覚フィードバック

## 技術スタック

- **React Native** 0.81 + **Expo SDK 54**
- **TypeScript** 5.9
- **NativeWind 4** (Tailwind CSS for React Native)
- **Expo Router 6** (ファイルベースルーティング)
- **Expo Audio** (音声再生)
- **React Native Reanimated 4** (アニメーション)
- **AsyncStorage** (ローカルデータ保存)

## セットアップ

### 1. 依存関係のインストール

```bash
pnpm install
```

### 2. 音声ファイルの配置

音声ファイルは容量の関係でリポジトリに含まれていません。`assets/audio/README.md` を参照して、音声ファイルを配置してください。

### 3. 開発サーバーの起動

```bash
pnpm dev
```

### 4. アプリの実行

#### iOS/Android実機でテスト

1. Expo Goアプリをインストール
2. QRコードをスキャン

#### Webブラウザでテスト

開発サーバー起動後、自動的にブラウザが開きます。

## ゲームの遊び方

1. **ホーム画面**: 「PLAY」ボタンをタップ
2. **難易度選択**: Easy、Normal、Hardから選択
3. **ゲームプレイ**: 
   - 画面上部から落ちてくるノーツを、下部のタップエリアに到達したタイミングでタップ
   - タイミングが良いほど高得点（Perfect > Good > Miss）
   - コンボを繋げてスコア倍率を上げる
4. **リザルト**: スコアと統計を確認し、リトライまたはメニューに戻る

## ゲームシステム

### 判定システム

- **Perfect**: ±50ms以内 → 100点 + コンボボーナス
- **Good**: ±100ms以内 → 50点 + コンボボーナス
- **Miss**: 判定範囲外 → 0点、コンボリセット

### スコアリング

- 基本スコア: Perfect=100点、Good=50点
- コンボ倍率: 10コンボごとに+0.1倍（例: 20コンボで1.2倍）
- 最終スコア = 基本スコア × コンボ倍率

### ノーツ生成

各難易度でノーツの密度が異なります：

- **Easy**: 2拍に1回、2レーンのみ使用
- **Normal**: 1拍に1回、3レーンまで使用
- **Hard**: 0.5拍に1回、全4レーン使用
